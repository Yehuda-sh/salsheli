# 📁 Project Structure - Salsheli (סל שלי)
# File: project_structure.txt
# Description: מבנה תיקיות מלא של פרויקט Salsheli - אפליקציית ניהול קניות חכמה

---

## 📱 סקירה כללית
**Salsheli** - אפליקציית Flutter לניהול רשימות קניות, מעקב מחירים, סריקת קבלות ומלאי חכם.

**פלטפורמות נתמכות:** Android, iOS (Mobile Only)
**גרסת Flutter:** Stable Channel
**ארכיטקטורה:** Clean Architecture עם Layers

---

## 🏗️ ארכיטקטורה - Layers

```
┌─────────────────────────────────────┐
│     UI Layer (Screens/Widgets)     │
│    screens/* + widgets/*            │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│   State Management (Providers)      │
│   providers/* (ChangeNotifier)      │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│   Business Logic (Services)         │
│   services/*                         │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│   Data Layer (Repositories)         │
│   repositories/*                     │
└─────────────┬───────────────────────┘
              │
┌─────────────▼───────────────────────┐
│   Data Sources (Local/Future API)   │
│   Local Storage / Firebase           │
└─────────────────────────────────────┘
```

## 📝 הערות לפיתוח

1. **תיעוד חובה** - כל קובץ חדש חייב הערת תיעוד בראש

5. **Code Review** - השתמש ב-`CODE_REVIEW_CHECKLIST.md`
