# 📚 מסמכי הסבר - MemoZap

## 🎯 מה זה?

מסמכים פשוטים שמסבירים איך האפליקציה עובדת.
כל מסמך כולל:
- הסבר פשוט
- תרשימי זרימה
- דוגמאות עם משתמש בשם **יוגב**
- רשימת קבצים ומה הם עושים
- בעיות לבדיקה

## 🔄 זרימה מהירה - מה יוגב עושה?

```
הרשמה → דף הבית → יצירת רשימה → הוספת מוצרים
                                    ↓
                              הולך לחנות
                                    ↓
                              קנייה פעילה
                                    ↓
                              סיום + קבלה
                                    ↓
                              המזווה מתעדכן
```

## 📱 מבנה האפליקציה

```
MemoZap
├── 🏠 בית (Tab 0)
│   ├── רשימות פעילות
│   └── היסטוריה
│
├── 📦 מזווה (Tab 1)
│   └── מלאי בבית
│
├── 👥 קבוצות (Tab 2)
│   └── שיתוף עם אחרים
│
└── ⚙️ הגדרות (Tab 3)
    ├── פרופיל
    ├── העדפות
    └── התנתקות
```
