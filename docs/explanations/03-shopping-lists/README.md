# 📱 רשימות קניות

## 🎯 מה זה עושה?

זהו הלב של האפליקציה - ניהול רשימות קניות.
מאפשר ליוגב:
- ליצור רשימה חדשה
- לבחור סוג רשימה (סופרמרקט, אטליז, בית מרקחת...)
- להוסיף מוצרים לרשימה
- לסמן מוצרים שנקנו
- לשתף רשימה עם אחרים

│  יוגב ממלא:                                             │
│  • שם הרשימה: "קניות לסופ"ש"                            │

│  • מי יראה: משפחתית 👨‍👩‍👧                                │

## 📖 דוגמה מפורטת - יוגב יוצר רשימה

**הסיטואציה:** יוגב רוצה ליצור רשימת קניות לסופ"ש

**מה יוגב עושה:**
1. לוחץ "+ רשימה חדשה" בדף הבית
2. ממלא "קניות לסופ"ש" בשדה השם
3. בוחר סוג רשימה: 🛒 סופרמרקט
4. בוחר "משפחתית" (כל המשפחה תראה)
5. לא מכניס תקציב
6. לוחץ "צור רשימה"

**מה נשמר ב-Firebase:**
```
shopping_lists/רשימה_id:
  name: "קניות לסופ"ש"
  type: "supermarket"
  status: "active"
  household_id: "house_abc123"
  is_private: false
  created_at: "2024-01-15 10:00:00"
  created_by: "יוגב_id"
  items: []
```

## 🔄 עדכון בזמן אמת

**חובה לרשימות משותפות!**

כשיוגב ושרה עורכים את אותה רשימה:
- כל שינוי שמישהו עושה → מופיע מיד לשני
- לא צריך לרענן ידנית

**מה קורה אם שניהם עורכים את אותו מוצר באותו זמן?**
- האחרון ששומר "מנצח" (Last-Write-Wins)
- זה בסדר - במקרים רגילים זה לא יקרה הרבה

## 👥 הרשאות עורך - בקשות לאישור

כשמשתמש עם הרשאת "עורך" רוצה להוסיף פריט:

**הזרימה:**
1. עורך לוחץ "הוסף מוצר"
2. המוצר נשלח כ"בקשה ממתינה"
3. בעלים/מנהל רואה את הבקשה
4. בעלים/מנהל מאשר או דוחה

**איך זה נראה ברשימה (לבעלים/מנהל):**

```
┌──────────────────────────────────────────────────────────────┐
│  🛒 רשימת קניות - סופר שישי                                 │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ✅ חלב 3% - 2 יח'                                          │
│  □  גבינה צהובה                                             │
│                                                              │
│  ─────── ⏳ ממתין לאישור ───────                            │
│                                                              │
│  ⏳ שוקולד פרה  (דני ביקש)        [✓ אשר]  [✗ דחה]         │
│  ⏳ ביסלי גריל  (דני ביקש)        [✓ אשר]  [✗ דחה]         │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**מה קורה עם בקשות שלא טופלו?**
- אם סיימו קנייה בלי לאשר → לא מופיעים בקבלה
- אם אישרו וקנו → מופיעים בקבלה עם "נוסף ע"י דני"

## 🎉 מצב "מי מביא מה" (לאירועים)

כשיוצרים רשימת אירוע ובוחרים "מי מביא מה":

**איך זה עובד:**
1. כל משתתף יכול לסמן פריט שהוא מביא
2. ליד הפריט מופיע שם מי שסימן
3. פריטים שאף אחד לא סימן = "פנוי"

**איך זה נראה:**

```
┌──────────────────────────────────────────────────────────────┐
│  🎉 על האש אצל משה - מי מביא מה                             │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  🥩 בשר טחון 2 ק"ג                             יוגב         │
│  🌽 תירסים x6                                  שרה          │
│  🥖 פיתות x20                                  דני          │
│  🧅 בצל לגריל                                  (פנוי)       │
│  🍅 עגבניות                                    (פנוי)       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**נעילת רשימה:**
- בעלים/מנהל יכולים ללחוץ "נעל רשימה"
- אחרי נעילה → לא ניתן לשנות מי מביא מה
- כולם רואים את החלוקה הסופית

## ⚠️ דברים לבדוק

### עובד:
- ✅ יצירת רשימה עם שם ייחודי
- ✅ בחירת סוג רשימה
- ✅ הגדרת פרטיות (אישית/משפחתית/שיתוף)
- ✅ בחירת תבנית מוכנה
- ✅ הוספת מוצרים מהקטלוג
- ✅ סימון מוצרים שנקנו (נשארים במקום עם קו חוצה)
- ✅ חיפוש בתוך הרשימה
- ✅ קיבוץ לפי קטגוריה (מעל 10 פריטים)
- ✅ חישוב סכום כולל
- ✅ מחיקת פריט עם אישור ואפשרות ביטול
- ✅ הרשאות עורך - בקשות לאישור

### מה לא מטופל / דורש שיפור:
- ⚠️ **עריכה בו-זמנית** - האחרון דורס (Last-Write-Wins). זה בסדר לפי ההחלטה.
- ⚠️ **הגבלת פריטים** - צריך להוסיף מקסימום 200 פריטים ברשימה
- ⚠️ **עדכון בזמן אמת** - חובה לרשימות משותפות! חסר כרגע.
- ⚠️ **אין אינטרנט** - צריך להציג הודעה ולחסום עריכה
- ⚠️ **מצב "מי מביא מה"** - לא מומש עדיין
- ⚠️ **כפתור קיבוץ/ביטול קיבוץ** - המשתמש יחליט
